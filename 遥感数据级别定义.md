# **遥感数据级别定义**

## **NASA（以Landsat为例）数据级别表**

| 级别   | 名称           | 处理程度                | 主要用途                 |
| :----- | :------------- | :---------------------- | :----------------------- |
| **L0** | 原始数据产品   | 数据解包与格式重建      | 地面系统存档与处理       |
| **L1** | 辐射校正产品   | 辐射定标 + 系统几何校正 | 所有定量分析的基准数据   |
| **L2** | 地表反射率产品 | L1数据 + 大气校正       | 地表参数反演与科学分析   |
| **L3** | 专题科学产品   | L2数据 + 模型反演       | 高阶生物地球物理产品生成 |



## **ESA（以Sentinel-2为例）数据级别表**

| 级别    | 名称             | 处理程度                   | 主要用途                     |
| :------ | :--------------- | :------------------------- | :--------------------------- |
| **L0**  | 原始数据         | 数据解码与排序             | 地面处理系统输入             |
| **L1C** | 系统几何校正产品 | 辐射定标 + 正射校正(无GCP) | 可视化、浏览与基础分析       |
| **L2A** | 大气校正产品     | L1C数据 + 大气校正         | 地表反射率分析与时间序列研究 |

##  **中国遥感卫星标准（参考）**

| 级别    | 名称           | 处理程度                      | 主要用途       |
| :------ | :------------- | :---------------------------- | :------------- |
| **L0**  | 原始数据       | 未作任何处理，原始下行数据    | 存档、专业处理 |
| **L1A** | 辐射校正后数据 | 辐射校正+系统几何校正+RPC参数 | 大多数应用起点 |
| **L1B** | 几何校正后数据 | L1A+几何校正（无DEM）         | 快速出图       |
| **L2**  | 正射校正产品   | L1A+正射校正（使用DEM/GCP）   | 制图、GIS应用  |
| **L3**  | 专题产品       | 分类、融合等增值产品          | 特定应用       |
